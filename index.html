About Sol-Fi

Sol-Fi is an on-chain tracker and analytics dashboard for the Solana blockchain. We compile open performance data from public validators to help users monitor staking metrics, node uptimes, and historical reward rates.

Multi-Chain Uptime Analytics

Monitor public node performance statistics using SOL, USDT, or BTC indicators. The system aggregates consensus data from Solana, Ethereum, Binance Smart Chain, and TRON networks to track node health parameters.

How to Get Started ⚡

Frequently Asked Questions

Have questions? We have compiled responses to the most frequent inquiries from our active user base.

What is Sol-Fi?
Sol-Fi is a decentralized data dashboard that indexes uptime and reward metrics for public Solana validator nodes. Performance metrics are retrieved directly from on-chain consensus history.
How do I start tracking?
Simply sign up, configure your read-only profile address, select a node delegation performance profile, and verify your account configurations.
Are metrics updated in real-time?
Yes. The system automatically fetches consensus updates directly from validator RPC endpoints, updating the active graphs to match network states.
Is there a referral community tier?
Yes. Users can share invitation links to gain premium analytics tier benefits. View details on your profile settings page.
